# Event Time Table

An Angular application for displaying events in a weekly timetable format with multiple venues.

## Features

- Weekly calendar view with 7-day display
- Multiple venue support with tabbed interface
- 15-minute time slot intervals (9:00 AM - 6:00 PM)
- Event scheduling with automatic time slot spanning
- Built with Angular 20 and Angular Material

## Prerequisites

- Node.js (LTS version recommended)
- npm (comes with Node.js)

## Installation

```bash
npm install
```

## Development

Start the development server:

```bash
npm start
```

Navigate to `http://localhost:4200/`

## Build

Build for production:

```bash
npm run build
```
